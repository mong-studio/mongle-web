# Original MVP Plan

## Summary

- Build a browser-based pixel village focus/todo game MVP.
- Use Phaser, TypeScript, and Vite.
- Prioritize graphics quality while using free assets where licensing is clear.

## Graphics Direction

- Prefer CC0 or commercially usable free assets.
- Keep tile size consistent at 16x16 or 32x32.
- Use nearest-neighbor rendering and integer-feeling scaling.
- Aim for a dense, readable, cozy village composition.

## Key Features

- Top-down village map with forest, paths, houses, benches, props, flowers, and NPCs.
- Top-left focus timer.
- Top-right date/todo panel.
- Center selected entity card.
- Bottom NPC dialogue panel.
- 25-minute focus timer.
- Todo add/complete with localStorage.
- NPC/building click interactions.

## Scope Boundaries

- No copied reference screenshot assets.
- No login, server sync, multiplayer, or packaged mobile app.
- MVP should stay small enough to polish visually.
